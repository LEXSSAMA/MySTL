| 操作接口      | 功能                    | 适用对象     |
| :----------:  | :------:                | :----------: |
| size()        | 返回二叉树的规模        | 二叉树       |
| empty()       | 返回该树是否为空        | 二叉树       |
| root()        | 根节点                  | 二叉树       |
| parent()      | 父节点                  | 二叉树       |
| firstChild()  | 长子                    | 二叉树       |
| nextSibling() | 兄弟                    | 二叉树       |
| insert(i,e)   | 将e作为第i个孩子插入    | 二叉树       |
| remove(i)     | 删除第i个孩子(及其后代) | 二叉树       |
| traverse()    | 遍历                    | 二叉树       |
